# Digi Solutions – University Complaint Management System

**Final Year Project (BS Information Technology, 2022–2026)**
University of Chakwal

## Overview

Digi Solutions is an AI-powered University Complaint Management System designed to streamline complaint handling through a digital platform. The system enables students and faculty to submit and track complaints, departments to manage and resolve issues, and administrators to monitor performance through AI-driven feedback analysis.

## Key Features

* Complaint submission and real-time status tracking
* Role-based access for Students, Faculty, Departments, and Admins
* Automated complaint routing to relevant departments
* OTP-based password recovery
* Push notifications and feedback collection
* AI-powered sentiment analysis of user feedback
* Department performance and complaint reports

## Technology Stack

* **Mobile App:** Java, Kotlin, XML (Android)
* **Web Portal & Dashboard:** HTML, CSS, JavaScript
* **Authentication:** Firebase Authentication
* **Database:** Supabase (PostgreSQL)
* **AI/ML:** Python, CatBoost, VADER, TextBlob, FastAPI
* **Notifications:** Firebase Cloud Messaging

## AI Integration

The system analyzes feedback submitted after complaint resolution using NLP and Machine Learning techniques to classify user sentiment and generate insights for administrators.

## Testing

Successfully passed all major test cases, including authentication, complaint management, notifications, feedback analysis, reporting, and access control.
